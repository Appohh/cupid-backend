User Registration & Authentication
As a new user, I want to register with Cupid using my email and password.
As a user, I want to verify my email address upon registration.
As a user, I want to log in to Cupid using my email and password.
As a user, I want the option to reset my password if I forget it.
Profile Management
As a user, I want to create and customize my profile with details like bio, photos, and interests.
As a user, I want to edit my profile information at any time.
As a user, I want to upload multiple photos to my profile.
Matching & Swiping
As a user, I want to see potential matches based on my preferences and location.
As a user, I want to swipe left to pass or swipe right to like a profile.
As a user, I want to receive notifications for mutual likes.
As a user, I want to view profiles I've matched with.
Messaging
As a user, I want to chat with my matches in a real-time messaging interface.
As a user, I want the ability to block or report other users if needed.
As a user, I want to receive notifications for new messages.
Preferences & Settings
As a user, I want to set my preferences such as age range, distance, and gender.
As a user, I want to adjust my notification settings (email, push notifications, etc.).
As a user, I want to deactivate or delete my account if desired.
Privacy & Security
As a user, I want my personal information to be secure and not shared without my consent.
As a user, I want the option to hide my profile from specific users or from appearing in searches.
Premium Features*
As a user, I want access to premium features like unlimited swipes or advanced search filters.
As a user, I want to upgrade my account to a premium membership.
Admin Panel 
As an admin, I want to monitor and moderate user activity and reported profiles.
As an admin, I want to have the ability to ban users or take necessary actions against inappropriate behavior.
